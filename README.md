# SNS 反省トラッカー

SNS使用前の期待と、使用後の実際の満足度を記録・可視化するシンプルなWebアプリケーションです。

## 概要

このアプリは、SNS使用の習慣を見直すためのツールです。SNSを見る前に持つ「期待」と、実際に見た後の「満足度」のギャップを統計的に可視化することで、SNS使用の実態を客観的に把握できます。

## 主な機能

### 📝 記録機能
- **Step 1**: SNS使用前に期待度を記録（1-5のスケール）
- **Step 2**: SNS使用後に満足度を記録（1-5のスケール）
- 使用したSNSの種類も記録可能（Twitter/X, Instagram, Facebook, TikTok, YouTube等）

### 📊 統計・可視化
- **総記録数**: これまでに記録した回数
- **平均期待度**: SNSに対する平均的な期待値
- **平均満足度**: SNSから得られた平均的な満足度
- **平均ギャップ**: 期待と現実の差（期待 - 満足度）

### 📈 グラフ表示
- **期待 vs 現実**: 直近10件の期待度と満足度の比較
- **ギャップの推移**: 期待と現実のギャップの時系列推移

### 💡 インサイト機能
記録が3件以上になると、以下のような分析結果を表示：
- 期待と現実のギャップ分析
- 期待外れ率の計算
- SNS別の傾向分析
- 最近の傾向（改善/悪化）

### 📜 履歴表示
- 過去の全記録を時系列で表示
- 各記録の詳細（日時、SNS種類、期待度、満足度、ギャップ）を確認可能

## 使い方

1. **ブラウザで開く**
   ```
   index.htmlをブラウザで開く
   ```

2. **SNS使用前**
   - 見たいSNSを選択
   - 期待度を1-5で評価
   - 「SNSを見に行く」ボタンをクリック

3. **SNS使用後**
   - アプリに戻る
   - 実際の満足度を1-5で評価
   - 「記録する」ボタンをクリック

4. **統計を確認**
   - 「概要」タブで統計とグラフを確認
   - 「履歴」タブで過去の記録を確認
   - 「インサイト」タブで分析結果を確認

## 技術スタック

- **HTML5**: セマンティックな構造
- **CSS3**: モダンなグラデーションとアニメーション
- **Vanilla JavaScript**: フレームワーク不要のシンプルな実装
- **Chart.js**: グラフ描画ライブラリ
- **LocalStorage**: データの永続化（ブラウザ内）

## データ保存について

すべてのデータはブラウザのLocalStorageに保存されます。
- サーバーにデータは送信されません
- プライバシーは完全に保護されます
- ブラウザのキャッシュをクリアするとデータが消えます

## カスタマイズ

### SNSの種類を追加
`index.html`の`<select id="snsType">`部分を編集：

```html
<select id="snsType">
    <option value="あなたのSNS">あなたのSNS</option>
    <!-- 追加 -->
</select>
```

### 評価スケールの変更
現在は1-5のスケールですが、`index.html`と`app.js`を編集することで変更可能です。

## 期待される効果

1. **自己認識の向上**: 自分のSNS使用パターンを客観的に把握
2. **行動変容のきっかけ**: データに基づいた意思決定
3. **SNS依存の軽減**: 期待と現実のギャップを可視化することで、無意識のSNS使用を減らす
